# Drupal CMS template

Build template for Wodby 2.0 [drupal-php](https://github.com/wodby/service-drupal-php) service with defined pipeline for Wodby CI

This repo is syncing with https://git.drupalcode.org/project/cms.git (latest stable tag) and adds drush/drush package

For Drupal CMS docker image see [wodby/drupal-cms](https://github.com/wodby/drupal-cms)
